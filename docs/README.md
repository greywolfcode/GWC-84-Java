# GWC-84 Java Docs

These docs are built using [ProperDocs](https://github.com/ProperDocs/properdocs).


## Requirements

Building these docs requires ProperDocs and the readthedocs theme

```
pip install properdocs
pip install properdocs-theme-readthedocs
```


## Serving

properDocs can serve the site for local testing using `properdocs serve`

> [!NOTE]
> A warning will be issued about README.md whe building, this is fine and the site builds as intended
